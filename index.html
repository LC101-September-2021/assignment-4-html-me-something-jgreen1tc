Catalyst

You are the center of something great and terrible. Earn a persona point each time your actions cause a conflict between two powerful personas or organizations -BWGR pg 199

Community-driven MUD running The Burning Wheel

Catalyst is planned to be an automated GM app for The Burning Wheel tabletop roleplaying system. It allows users to host/join games of The Burning Wheel, allowing them easy access to their important game assets, while handling the complicated rules interpretations for them. It relies on community creation and input to generate the narrative content of the campaign. Follow the projects progress on GitHub.

Planned features/vision

The full feature-set of the project will ultimately be based upon the Burning Wheel Procedural Guide. A strong benefit of Burning Wheel is that while complicated, the rules tightly guide the experience in a player character focused way. I believe this is a great opportunity for software to manage the complexity of a task but keep the user in control of which task it handles. Gameplay should be intuitive so that the users can focus on the story they create with the assurance that they are playing the system both correctly and optimally.

    Users should be able to login and create:

    • Games
      • with the mechanics of GM/Player turns to be tracked
    • Characters
      • with the ability for character rolls and progression to be tracked
