The Catalyst

Community-driven MUD running The Burning Wheel

The Catalyst is planned to be an automated GM app for The Burning Wheel tabletop roleplaying system. It allows users to host/join games of The Burning Wheel, allowing them easy access to their important game assets, while handling the complicated rules interpretations for them. It relies on community creation and input to generate the narrative content of the campaign.

Planned features

    Users should be able to login and create:

    • Games
      • with the mechanics of GM/Player turns to be tracked
    • Characters
      • with the ability for character rolls and progression to be tracked

Initial task: Character sheet

Although there are many more features planned, the most natural place to start was getting an automated character sheet working. The first thing I made was the basic die rolling and the building blocks for each character element as a javascript class.
